Jhandi Burja

Published on: April 5, 2023 4 min read

Introduction

Jhandi Burja is a digital adaptation of the traditional Nepali board game, bringing this cultural treasure to the digital world. This web-based game allows players to enjoy the classic game online with friends or against AI, complete with authentic rules and interactive gameplay.

Project Objectives

  • Digitally recreate the traditional Jhandi Burja board game with authentic rules
  • Develop an intuitive user interface for both desktop and mobile devices
  • Implement both single-player (vs AI) and multiplayer modes
  • Add sound effects and animations for an engaging gaming experience
  • Ensure the game is accessible and easy to learn for new players

Technical Implementation

Built entirely with frontend technologies for fast performance and smooth gameplay. The game features responsive design, touch controls for mobile devices, and local storage for saving game progress.

Key Technologies:

  • Frontend: HTML5, CSS3, JavaScript (ES6+)
  • Game Engine: Custom game logic implementation
  • UI/UX: Responsive design with mobile-first approach
  • Animation: CSS3 animations and transitions
  • Deployment: Static hosting on Netlify
